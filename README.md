визуал:

1. главная

поле ввода айди
войти (получить талон)
ниже кнопка создать комнату
в углу текущий айди слепок полупрозрачный

2. комната (вид юзера)

поцентру талон
справа от талона информация: текущее место в очереди, всего человек, время текущего обслуживания, время всей очереди (может еще метрики какие-то полезные)
кнопка выйти

3. админ панель

карточки комнат
кнопка создать комнату
при клике на карточку открывается страница детализации

4. комната (вид админа)

список очередей
кнопка создать/удалить очередь
карточки талонов (драг н дроп присутствует)
текущий талон
кнопка следующий/завершить (если есть текущий талон - завершить, если статус ожидания - следующий, заменяются)

# Юзкейсы

юзер:

1. взять талон

юзер заходит на главную - вводит айди комнаты - нажимает получить талон: 1. комната есть - автораспределение в очередь (балансировщик) 2. очереди нет - выходит юай ошибки

2. выйти 

юзер в комнате нажимает выйти - его удаляют из очереди досрочно

админ:

1. создать комнату

на галвной нажимает создать комнату любой человек - он становится админом и его перекидывает на пустую панель - отображаются все текущие комнаты по данном айди - нажимает кнопку создать комнату - заполнение формы - переадресация на детализацию комнаты

2. распределенные комнаты

одна комната может иметь несколько очередей, каждая очередь буква в алфавите (А, B, C) и номер талона. при заходе в комнату юзеров происходит балансировка очередей (добавляется в меньшую очередь)

3. управление очередью

очередь в состоянии ожидания - админ нажимает вызвать следующего - попается самый старый талон - фаза активной очереди

активный талон - кнопка завершить - статус ожидания
